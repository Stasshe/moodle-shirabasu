使い方
---

事前準備:

1. Python3 と pip を用意してください。
2. 依存をインストール:

```bash
pip install -r requirements.txt
```

基本的な使い方:

- `./in/codes.txt` に1行1コードで授業コードを書く:

```
53317
202653317
```

実行:

```bash
python main.py
```

オプション:

- `--open` : 取得後にブラウザで開く。
- 先頭に授業コードを1つだけ渡すと、そのコードだけ単発処理できます。

出力:

- 実行ごとに `out/syllabus_<授業コード>.json` が作成されます。抽出される主なフィールドは `title`, `teachers`, `fields`, `tables`, `lists` です。

注意:

- ページによっては JavaScript で描画されるため Selenium（ChromeDriver 等）の準備が必要です。Selenium がない場合は単純取得に失敗する可能性があります。
- 公開ページのみを対象に、個人利用の範囲で利用してください。
